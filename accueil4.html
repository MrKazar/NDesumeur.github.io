Liste des Jeux

Bienvenue sur la page de jeux. Vous pouvez trouver ci-dessous une liste de jeux disponibles.

#Multiplatforme Snake

Snake est un jeu classique dans lequel vous contrôlez un serpent qui se déplace sur un plateau. Le serpent peut traverser les murs et réapparaît de l'autre côté. Le but est de manger la nourriture et de devenir le plus long possible sans heurter vous-même. Plus vous mangez de la nourriture, plus le serpent devient long et le jeu s'accélère.

#Multiplatforme 2048

2024 est un jeu de puzzle dans lequel vous combinez des tuiles numérotées pour atteindre la tuile "2048". Le jeu se déroule sur une grille 4x4, et vous pouvez déplacer les tuiles dans quatre directions pour les fusionner. Le but est d'obtenir le plus grand score sans être bloqué (grille pleine et aucune case qui peut se fusionner).

#Multiplatforme Colors

Colors est un jeu de mémoire dans lequel vous devez reproduire une séquence de couleurs de plus en plus longue. Chaque couleur s'illumine dans un ordre aléatoire, et vous devez les sélectionner dans le même ordre pour continuer. Le jeu devient de plus en plus difficile à mesure que vous réussissez à reproduire des séquences plus longues.
